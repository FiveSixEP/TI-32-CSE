# TI-32

A fork to give TI-84 Plus C Silver Edition calculators access to Gemini API (or OpenAI API) chat completions.

![built pcb](./pcb/built.png)

## Fork differences

- Gemini API support
- Support for multi-page response from GPT

## Documentation

Read the DOCUMENTATION.md file
